# Godot-3DSample

A 3D game developed using **Godot 4**, created by following structured tutorials and enhanced with enemy AI, pathfinding, and animation systems.

- LegionGames – *3D Enemies With Pathfinding and Animations (Godot 4 FPS Tutorial)*
  https://youtu.be/iV710Vm5qm0?si=N0kBm7fFsZffhmnt
